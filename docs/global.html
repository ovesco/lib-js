CommonMeta

Common Meta are returned by each standard call on the API https://api.pryv.com/reference/#in-method-results

Properties:
Name Type Description
apiVersion string

The version of the API in the form {major}.{minor}.{revision}. Mirrored in HTTP header API-Version.

The current server time as a timestamp in second. Keeping track of server time is necessary to properly handle time in API calls.
serial string

The serial will change every time the core or register is updated. If you compare it with the serial of a previous response and notice a difference, you should reload the service information.
